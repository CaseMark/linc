# Contributing to Linc

Thanks for your interest in contributing.

## Before You Start

1. Open an issue describing what you want to change and why
2. Wait for a maintainer to approve before submitting a PR

## Requirements

```bash
npm run check  # must pass with no errors
```

If you use an AI coding agent, run it from the repo root so it picks up `AGENTS.md`.

## Philosophy

Linc's core is minimal. If your feature can be an extension, it should be. See [docs/extensions.md](packages/coding-agent/docs/extensions.md).

## Questions

Open an issue on [GitHub](https://github.com/CaseMark/linc/issues).

# Development

Clone the repository:

```bash
git clone https://github.com/CaseMark/linc
cd linc
npm install
```

Run from source:

```bash
npx tsx packages/coding-agent/src/cli.ts
```

Run checks:

```bash
npm run check
```

Build the public docs:

```bash
npm run docs:build
```

Preview the public docs locally:

```bash
npm run docs:preview
```

Linc is based on Pi and selectively tracks upstream changes while keeping case.dev as the provider path.
